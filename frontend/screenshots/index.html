Getting Started

Platform Overview

Hada Commerce gives you everything you need to run an online store: product management, order processing, customer tracking, content pages, promotions, themes, plugins, and an AI assistant — all in one platform.

Product Management

Create and manage products with variants, pricing, inventory tracking, and rich descriptions. Organize with categories and collections.

Order Processing

Track orders from placement to delivery with status management, customer details, and order item breakdowns.

Customer Management

View customer profiles, order history, and manage your customer base with search and filtering.

Block-Based Pages

Build storefront pages with a drag-and-drop block editor. Use pre-built blocks like hero banners, product grids, and CTAs.

Theme System

Design your store's look with theme tokens: colors, typography, spacing, borders, and shadows. Switch themes instantly.

Plugin Marketplace

Extend your store with plugins for reviews, email campaigns, analytics, and more. Plugins hook into events via webhooks.

AI Agent

Chat with an AI assistant that can create products, manage orders, update settings, and answer questions about your store.

Headless API

Full REST API for building custom storefronts with any framework. Public and authenticated endpoints for every resource.

Customer-Facing

Storefront

Your customers see a polished storefront with product browsing, category navigation, a shopping cart, and checkout — all rendered from your block-based pages and theme tokens.

Home Page

The storefront home page displays your hero banner, featured products, category navigation, and a call-to-action section. All content is driven by the block editor.

  • Hero banner with customizable headline and CTA buttons
  • Featured products grid with pricing and add-to-cart
  • Category cards for easy browsing
  • Trust badges (free shipping, returns, etc.)
  • Footer call-to-action section

Product Listing

Customers can browse all products with pricing, stock status, and descriptions. Each product shows its SKU, price, and an add-to-cart button.

  • Grid layout with product cards
  • Price display with currency formatting
  • Stock status indicators
  • Quick add-to-cart functionality

Shopping Cart

Customers can review their cart with item quantities, individual pricing, and order totals before proceeding to checkout.

  • Quantity adjustment controls
  • Per-item and subtotal pricing
  • Remove items from cart
  • Proceed to checkout button

Checkout

A streamlined checkout flow collects shipping information and displays the order summary with all items and totals.

  • Shipping address form
  • Order summary sidebar
  • Payment integration ready
  • Order confirmation
Security

Authentication

Hada Commerce uses OAuth-based authentication with Google and Microsoft. Admins sign in to access the dashboard and manage their store.

Admin Login

The admin login page provides a clean, branded sign-in experience with OAuth provider buttons. No passwords to manage.

  • OAuth with Google and Microsoft
  • Branded login card with store logo
  • JWT-based session management
  • Automatic redirect after authentication
Multi-Tenant Security

Every API request is scoped to the authenticated tenant. Data isolation is enforced at the database query level — tenants can never access each other's data.

Organize your products into categories and collections. Categories define the product taxonomy, while collections group products for marketing and merchandising.

Categories & Collections

Manage your product taxonomy with categories (hierarchical) and collections (curated groups). Products can belong to multiple categories and collections.

  • Create and edit categories with slugs
  • Build curated product collections

Content

Pages & Block Editor

Build storefront pages visually with the block editor. Choose from 10+ block types including hero banners, product grids, testimonials, and more. Each page can be HTML or block-based.

Page Management

View all storefront pages with their title, slug, status (Published/Draft), content type (HTML/Blocks), and last update date. Create new pages with either the block editor or raw HTML.

  • Published/Draft status management
  • HTML or Block content types

Block Types

Available block types: Hero, Rich Text, Product Grid, Featured Collection, Image, Video, Banner, Newsletter Signup, Testimonials, and CTA. Each block type has configurable settings for content, styling, and layout.

Design

Theme System

Control your store's visual identity with design tokens. Create multiple themes with custom colors, typography, spacing, borders, and shadows. Activate any theme instantly.

Theme Editor

Create and manage themes with a full design token editor. Each theme defines colors (primary, secondary, accent, background, text, border), typography (font families, sizes, weights), spacing scales, border radiuses, and shadow levels.

  • Color palette preview on each theme card
  • Active theme indicator
  • Copy themes as a starting point
  • One-click theme activation
  • Full token editor with live preview
CSS Custom Properties

Theme tokens are converted to CSS custom properties (e.g., --hada-color-primary, --hada-font-heading) and injected into the storefront renderer. Your store's design updates instantly when you switch themes.

Extensions

Plugin Marketplace

Extend your store's functionality with plugins. Browse the marketplace, install plugins, and configure them to add features like reviews, email campaigns, and advanced analytics.

Browse & Install Plugins

The marketplace shows available plugins with descriptions, categories, tags, and installation status. Browse all plugins or filter by category (Official, Community).

  • Browse with search and category filters
  • Plugin cards with name, author, description, and tags
  • One-click install/uninstall
  • Installed count badge
  • Enable/disable installed plugins
Plugin Architecture

Plugins receive events via webhooks (order placed, product created, customer registered, etc.) and can inject JavaScript into the storefront via the JS manifest endpoint. This enables real-time integrations without modifying core code.
